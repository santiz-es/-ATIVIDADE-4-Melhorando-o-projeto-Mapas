# Flutter Mapas OSM

[ATIVIDADE 4] Melhorando o projeto Mapas
Considerando o projeto Mapas apresentado em sala, substitua o ícone padrão do Flutter. Você escolhe o ícone da sua aplicação. Além disso, adicione ao código:

Barra de busca de endereço
Atualização da posição no mapa durante a movimentação do dispositivo

## Funcionalidades

- Mostrar a localização atual do dispositivo em um mapa.
- Atualização em tempo real da posição do usuário.
- Barra de busca para localizar endereços pelo nome.
- Ícone de aplicativo personalizado.

---

## Pacotes Utilizados

- [`flutter_map`](https://pub.dev/packages/flutter_map) → Exibe o mapa OpenStreetMap.
- [`latlong2`](https://pub.dev/packages/latlong2) → Manipulação de coordenadas geográficas.
- [`provider`](https://pub.dev/packages/provider) → Gerenciamento de estado.
- [`geolocator`](https://pub.dev/packages/geolocator) → Obtenção da localização do dispositivo.
- [`geocoding`](https://pub.dev/packages/geocoding) → Conversão de endereço em coordenadas.
- [`battery_plus`](https://pub.dev/packages/battery_plus) → Ajuste de precisão baseado na bateria.
- [`flutter_launcher_icons`](https://pub.dev/packages/flutter_launcher_icons) → Geração de ícones personalizados.
